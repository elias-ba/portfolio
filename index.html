The Challenge

My Solution

WhatsApp-Integrated Inventory Management for Informal Retail

Computer vision solution for West African shop owners with basic smartphones

Client: Lengo AI References: Max Smith, CEO (max@lengo.ai) | Ismaila Seck, CTO (ismaila@lengo.ai)

The Challenge

Small shop owners in West Africa need inventory management but lack resources for traditional systems. They have smartphones with poor cameras, unreliable internet, and no technical training.

My Solution

  • Built OCR pipeline resilient to blur, poor lighting, and off-angle captures using TrOCR
  • Developed YOLO-based object detection trained on 50,000+ images of African retail products
  • Created WhatsApp bot interface allowing shopkeepers to send photos and receive inventory counts
  • Implemented edge processing to minimize bandwidth usage
2,000+ shops using the system across Senegal

Key Innovation

Traditional computer vision fails in these environments. I developed a multi-stage preprocessing pipeline that enhances image quality before analysis, and trained models specifically on low-quality footage. The system learns from corrections, improving accuracy for each shop's specific inventory over time.

Technologies Used

Python PyTorch YOLO v5 TrOCR HuggingFace Llama OpenCV WhatsApp Business API Redis AWS S3

Automated Customs Data Extraction System

Transforming manual logistics workflows at CMA CGM

Client: CMA CGM Reference: Fode Sy, CTO (DKA.FSY@cma-cgm.com)

The Challenge

CMA CGM staff manually processed hundreds of Bill of Lading forms daily from Senegal's Gaïndé customs portal. Each form took 15-20 minutes to process, creating bottlenecks and human errors in international shipping operations.

My Solution

  • Reverse-engineered the Gaïndé portal's JavaScript-heavy interface
  • Built Selenium-based scraper handling dynamic content and CAPTCHA challenges
  • Created intelligent parsing system for inconsistent form formats
  • Developed Excel generation pipeline matching CMA CGM's internal workflows
80% reduction in manual data entry time

Technical Challenges Overcome

The customs portal used aggressive anti-bot measures. I implemented human-like interaction patterns, distributed scraping across time windows, and built a robust error recovery system. The parser handles 47 different form variations using regex patterns and fuzzy matching.

Technologies Used

Python Selenium WebDriver BeautifulSoup4 Pandas OpenPyXL Regular Expressions Cron Jobs
